# Tirocinio - Player Tracking & Pose Estimation

## Summary
This repository contains a video processing pipeline designed for player tracking and pose estimation in team sports. It integrates **YOLOv11** for object detection and pose estimation, **SAHI (Slicing Aided Hyper Inference)** for handling high-resolution imagery, and a custom **Hungarian-based multi-cue tracker** that utilizes IoU, appearance descriptors (HSV histograms and LAB color), and pose vectors.

## Structure
- **Root Directory**: Contains the core Python scripts for video processing, tracking logic, and utility functions.
- **`Tracking/`**: Stores project documentation, presentation materials, and datasets (videos, rectified videos, and ground truth data).
- **`.zencoder/` / `.zenflow/`**: CI/CD and workflow configurations.

## Language & Runtime
**Language**: Python  
**Version**: Python 3.8+ (required by `ultralytics` and `sahi`)  
**Build System**: Script-based execution  
**Package Manager**: `pip`

## Dependencies
**Main Dependencies**:
- `ultralytics`: YOLOv11 implementation for detection and pose estimation.
- `sahi`: Slicing Aided Hyper Inference for small object detection.
- `opencv-python` (`cv2`): Image and video processing.
- `numpy`: Numerical operations and matrix handling.
- `scipy`: Optimization tools (specifically `linear_sum_assignment` for the Hungarian algorithm).

## Build & Installation
No formal build process is required. To set up the environment, install the following dependencies:

```bash
pip install ultralytics sahi opencv-python numpy scipy
```

## Main Files & Resources
- **`process_video_two_views.py`**: Main entry point for processing videos from two different camera views simultaneously.
- **`process_video.py` / `process_image.py`**: Scripts for single-view video and image processing.
- **`iou_tracker.py`**: Core tracking implementation using the Hungarian algorithm and multi-cue matching.
- **`appearence_utils.py`**: Utilities for calculating team appearance descriptors and pose vectors.
- **`yolo11_pose.py`**: Standalone script for YOLOv11 pose estimation with ROI masking.
- **`Process_Video_Pipeline.md`**: Detailed documentation of the video processing pipeline steps.

## Testing & Validation
- **Validation**: No formal testing framework (like `pytest`) is configured. Validation is performed through manual inspection of output videos and logs generated by scripts like `diagnose_alignment.py` and `Python_test.py`.
- **Quality Checks**: The pipeline includes various filtering steps (confidence thresholds, IoU thresholds) to ensure detection quality.
